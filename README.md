# Dev.Finances

Trata-se de uma aplicação de um sistema de controle financeiro, onde o usuário pode inserir seus ganhos e gastos, especificando o valor, descrição e a data referente ao item. As informações inseridas na aplicação são salvas automaticamente e diretamente no navegador por meio do LocalStorage.


### Caracteristicas

- A aplicação foi desenolvida utilizando HTML5,  CSS3 E JavaScript.
- As Informações são salvas no LocalStorage.
